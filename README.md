# Playstation-Symbols-LED
English:

Arduino Nano Code for the Playstation Symbols with two Buttons for Brightness and Effects

Effects

PS1 Boot Animation – Startup animation with smooth glow-in, short white flash and gentle fade-out.

Smooth Breathe – All symbols breathe together in a 4-second cycle.

Color Bounce – Colors rotate forward and backward through the symbols.

Sparkle White – White sparkling across all LEDs with smooth fading.

Sparkle Segmented – Sparkling stays within the fixed symbol colors (Pink, Blue, Red, Green).

Slow Fade Chase – Symbols softly fade in and out one after another with smooth crossfade transitions.


Long Press Effect Button (3sec) switches to Night Mode and back

Deutsch/German:

Arduino Nano Code für die Playstation Symbole mit zwei Push Buttons für Helligkeit und Effekte

Effekte

PS1 Boot Animation – Startanimation mit Glow-In, weißem Blitz und sanftem Fade-Out.

Smooth Breathe – Alle Symbole atmen gemeinsam in einem 4-Sekunden-Zyklus.

Color Bounce – Farben rotieren vorwärts und rückwärts durch die Symbole.

Sparkle White – Weißes Funkeln über alle LEDs mit sanftem Ausfaden.

Sparkle Segmented – Funkeln innerhalb der festen Symbolfarben (Pink, Blau, Rot, Grün).

Slow Fade Chase – Symbole faden weich mit Überblendung nacheinander ein und aus.


Langer Druck auf den Effekt Button (3sek) schaltet den Nachtmodus an oder aus
